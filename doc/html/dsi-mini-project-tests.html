Documentation for "dsi-mini-project-tests" tool

Description

The "dsi-mini-project-tests" tool uses the embedded H2 RDBMS engine (which supports a subset of SQL standard) to execute given queries on given datasets, and then compares the resulting relations with the ones produced by the "dsi-mini-project" tool.

Usage

To run the tests on both datasets ("dataset1-uniform" and "dataset2-zipf"), all queries ("query1" to "query3"), all scales ("scale1" to "scale6") and all algorithms ("sortmerge", "sortmergetrie", "leapfrog"), execute the command:

java -jar dsi-mini-project-tests.jar <path to the dataset> <path to dsi-mini-project executable>,

for example java -jar dsi-mini-project-tests.jar ../../dsi-mini-project/data/ ../../dsi-mini-project/bin/dsi-mini-project.
