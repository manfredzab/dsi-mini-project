Detailed Description

Linear iterator class based on the binary search tree.

It extends the simple iterator (

See Also
SimpleIterator) by providing a Seek() method which finds a seek key in O(log N) time and O(1 + log N/m) amortized time, where N is the number of nodes in the relation and m is the number of visits to keys in ascending order.
