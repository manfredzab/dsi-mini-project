Detailed Description

Multi-way sort-merge join iterator.

It is based on an abstract multiway sort-merge join iterator, with the main difference being that it uses the linear iterator portions of the trie-based trie iterator to linearly seek the iterator (by repeatedly calling Next()) iterator to the next candidate key.
